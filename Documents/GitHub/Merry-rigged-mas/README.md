A fun Christmas lights voting app for neighborhoods to vote for their favorite holiday displays!

## Features
- Vote for your favorite Christmas light displays
- View real-time voting results
- Leave anonymous messages and compliments
- Fully responsive design
- Secure voting with rate limiting

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser
